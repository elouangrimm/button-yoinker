# Button Yoinker

Button Yoinker is a Chrome browser extension that "yoinks" buttons from the websites you visit.
Do your usual everyday online stuff and watch the collection of your yoinked buttons grow. It’s fun, useless, and free!

You can install Button Yoinker by loading the extension unpacked from directly from the code of this repo.

Button Yoinker works locally and **doesn’t send data anywhere**, so your data remains private.

----

This extension is an enhancment to [Anatoly Zenkov](https://github.com/anatolyzenkov)'s original prokect [Button Stealer](https://github.com/anatolyzenkov/button-stealer). Thanks to him for the great idea and original code!
